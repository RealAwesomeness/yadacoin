Our Features

YadaCoin is a social media protocol that pays for a itself using it's own crypto currency.

User authentication

Using open standards, YadaCoin enables instantaneous off-chain user authentication.

Open relationships

Make a friend on a YadaCoin enabled site and you are able to share that relatinship on other YadaCoin enabled sites.

inter-organizational

Because the user relationships are stored on a blockchain, inter-organizational interactivity is possible, supplanting email.

Privacy

Unique user identifiers are not stored on the blockchain, so outsiders cannot see your organization.

Security

YadaCoin uses well-established open standards to secure its network and protocol.
